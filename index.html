Caros amigos, o Meu1ºSite foi desenvolvido utilizando o Bootstrap que tem como objetivo tornar mais fácil o desenvolvimento de interface (front-end) para páginas web. Ele disponibiliza padrões para os elementos HTML mais usados, além de elementos personalizados com o usado de elementos e classes css padrões..
